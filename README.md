---

## 🏛️ Profil Singkat  
IAIN Palopo sebagai kampus pencetak **pemimpin visioner** terus mendorong mahasiswa untuk:  
✅ Mengembangkan proyek berbasis *Islamic Values*  
✅ Menjadi problem solver melalui pendekatan multidisiplin  
✅ Membangun jejaring dengan komunitas global  

---

## 🚩 Program Unggulan

### 🔍 **Riset Terapan**
- **Smart Tahfidz**: Sistem hafalan Al-Qur'an dengan AI *voice recognition*  
- **Eco-Masjid**: Konsep masjid ramah lingkungan dengan panel surya  
- **Digital Baituzzakah**: Platform zakat/infaq real-time analytics  

### 🌱 **Pengabdian Masyarakat**
- **Desa Digital**: Pelatihan UMKM berbasis e-commerce untuk 15 desa  
- **Taman Baca Qurani**: Membangun 20 perpustakaan mini di pelosok Luwu  
- **Dakwah Visual**: Produksi konten kreatif Islami (animasi, infografis)
